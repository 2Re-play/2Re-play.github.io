## Vue.JS 설치



### CDN으로 설치하기

```html
<script src="https://unpkg.com/vue"></script>
```



### NPM으로 설치하기

```shell
# 최신 stable 버전
$ npm install vue
```


## 개발 도구 설치


### Vue CLI

```shell
# vue-cli 설치
$ npm install --global vue-cli
# "webpack" 템플릿을 이용해서 새 프로젝트 생성
$ vue init webpack my-project
# 의존성을 설치하고 실행하세요!
$ cd my-project
$ npm install
$ npm run dev
```
